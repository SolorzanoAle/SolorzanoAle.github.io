DESCRIPCION

La propiedad text-shadow agrega sombras al texto. Acepta una lista de sombras separadas por comas para aplicar al texto y cualquiera de sus archivos decorations. Cada sombra se describe mediante una combinación de compensaciones X e Y del elemento, el radio de desenfoque y el color.

Esta propiedad se especifica como una lista de sombras separadas por comas. Cada sombra se especifica como dos o tres length valores, seguido opcionalmente por un color valor. Los dos primeros length valores son los valores offset-x y offset-y. El tercer valor, opcional, length es el blur-radius. El color valor es el color de la sombra.

VALORES

COLOR

Opcional. El color de la sombra. Se puede especificar antes o después de los valores de compensación. Si no se especifica, el valor del color se deja en manos del agente de usuario, por lo que cuando se desea coherencia entre navegadores, debe definirlo explícitamente.

offset-x y offset-y

Requerido. Estos valores especifican la distancia de la sombra desde el texto.

offset-x especifica la distancia horizontal; un valor negativo coloca la sombra a la izquierda del texto.

offset-y especifica la distancia vertical; un valor negativo coloca la sombra sobre el texto.

Si ambos valores son 0, la sombra se coloca directamente detrás del texto, aunque puede ser parcialmente visible debido al efecto de blur-radius.

blur-radius

Cuanto mayor sea el valor, mayor será el desenfoque; la sombra se vuelve más ancha y más clara. Si no se especifica, el valor predeterminado es 0.

EJEMPLOS

LISTA DE SOMBRAS SEPARADAS POR COMA

              CODIGO: 
               text-shadow: 1px 1px 20px black, 0 0 1em blue, 0 0 0.2em blue;
               1px:distancia-horizontal
               1px:distancia-vertical
               20px:desenfoque-sombre
               COLOR black: recordemso que el color puede ir de primero o de ultimo
