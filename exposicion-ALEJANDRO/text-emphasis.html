TEXT-EMPHASIS

Esta propiedad es una abreviatura de las siguientes propiedades

  1. text-emphasis-color
  2. text-emphasis-style

Esto quiere decir que los valores que recibira son el color y el estilo en cualquier orden.

Valores de estilo que puede recibir:

  • FILLED
  • OPEN
  • CIRCLE
  • DOUBLE-CIRCLE
  • SESAME
  • TRIANGLE
  • DOT

CODIGO 
                .e1{
                    text-emphasis: red filled;
                }
                .e2{
                    text-emphasis: red open;
                }
                .e3{
                    text-emphasis: red circle;
                }
                .e4{
                    text-emphasis: red double-circle;
                }
                .e5{
                    text-emphasis: red sesame;
                }
                .e6{
                    text-emphasis: red triangle;
                }
                .e7{
                    text-emphasis: red dot;
                }

Tambien podemos agreagr una letra o simbolo que nosotros queramos metiendo ese valor entre comillas ejemplo:

text-emphasis: 'x';

EJEMPLO CON X

Su valor predeterminado es NONE
