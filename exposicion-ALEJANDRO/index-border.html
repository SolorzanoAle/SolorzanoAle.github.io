BORDER-SPACING

Descripcion

El border-spacing propiedad establece la distancia entre los bordes de las celdas adyacentes.

Para que la propiedad border- spacing sirva la propiedad BORDER-COLLAPSE tiene que estar en SEPARATE, aunque su valor por defecto es SEPARATE

EJEMPLO

                #table1 {
                    border-collapse: separate;
                    border-spacing: 15px;
                  }
                  
                  #table2 {
                      border-collapse: separate;
                    border-spacing: 20px 50px;
                  }

EXPLICACION BORDER-SPACING

La propiedad BORDER-SPACING recibe dos parametros el primer parametro es la distancia entre celdas(columnas) y el segundo parametro es la distancia entre FILAS

NOTA:

Si solo se deja el primer parametro el espacio entre filas y columnas sera ese parametro que recibio

Esta propiedad se le aplica a los bordes de una tabla

SINTAXIS

border-spacing: length|initial|inherit;

Si se deja en initial establece la propiedad predeterminada 2px

Si se deja en inherit hereda de su elemento padre
