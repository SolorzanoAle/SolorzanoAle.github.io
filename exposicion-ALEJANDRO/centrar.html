DESCRIPCION

Para centrar una imagen en html con CSS hay que recordar que la etiqueta IMG es una etiqueta en linea

Hay varias formas de centrar una imagen en la pantalla la primera de ellas es:

Añadiendo la etiqueta IMG a una etiqueta en bloque como la etiqueta p

EJEMPLO

              p{
                text-align: center;
                
            }

Si la etiqueta que en la que añadistes la imagen no es una etiqueta en bloque se deberia de usar la propiedad DISPLAY:BLOCK

Pero de igual manera usamos la propiedad text-align como en el siguiente ejemplo

EJEMPLO

              span{
                display: block;
                text-align: center;
            }

Otra forma es a la etiqueta IMG pasar a una etiqueta en bloque y ponerle un margin en auto

EJEMPLO

              .FRESAS{
                display: block;
                margin: auto;
            }

Tambien se puede usar la etiqueta CENTER de HTML
