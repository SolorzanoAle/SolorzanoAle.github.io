DESCRIPCION

La propiedad abreviada list-style le permite establecer todas las propiedades de estilo de lista a la vez.

Debido a que esta propiedad se hereda, se puede establecer en un elemento principal (normalmente ol o ul ) para que el mismo estilo de lista se aplique a todos los elementos que contiene.

            Esta propiedad es una abreviatura de las siguientes propiedades CSS:

            list-style-image
            list-style-position
            list-style-type

HEREDA:

Si

Valores iniciales

  • list-style-image= Disc
  • list-style-position= outside
  • list-style-type= none

LIST-STYLE esta propiedad es una abreviatura de otras 3 propiedades y recibe los parametros en el orden que sea.

EJEMPLOS

  1. Aca cambiamos el tipo de viñeta
    2. list-style: square;
  2. Aca asignamos como viñeta una imagen
    3. list-style: url("../ima-ina.jpg");
  3. Aca asignamos la posicion que tendra nuestra lista
    4. list-style: inside;
  4. Aca asignamos el tipo de viñeta que queremos que tenga la lista y la posicion.
    5. list-style: georgian inside;
  5. Aca asignamos dos tipos de viñetas ya que si no encuentra la imagen se pondra la viñeta lower-roman
    6. list-style: lower-roman url("../img/shape.png") outside;
